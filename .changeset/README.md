# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can find the
full documentation for it [in the readme](https://github.com/changesets/changesets/blob/main/README.md).

## Usage

To create a new changeset, run `pnpm changeset` in the root of the repository. This will prompt you to select
which packages are affected and whether the change is a major, minor, or patch bump.

The generated changeset file should be committed with your pull request. When the PR is merged, the release
workflow will automatically create a "Version Packages" PR that bumps versions and updates changelogs.
